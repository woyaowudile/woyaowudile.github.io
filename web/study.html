# 三. 样式设置

使用审查元素找到需要修改样式的元素

然后在styles文件夹下的palette.styl中

.page {
   > div {
       > blockquote {
           word-break: break-word;
       }
   }
   .theme-default-content {
       .line-numbers-mode {
           .line-numbers-wrapper {
               width: 2.5rem;
           }
           &::after {
               width: 2.5rem;
           }
       }
   }
}
